# LibTuner
An Arduino library to control Rafael Micro tuners like the R820T.

## Description
This library is a heavily modified version of the librtlsdr https://github.com/librtlsdr/librtlsdr for the purpose of porting to Arduino the part of code that controls the Rafael Micro chips.

This is the EXPERIMENTAL version. It doesn't work yet!
